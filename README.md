# Subscriptions API
Simple subscriptions service

You can run this app with:
```
docker compose up
```
You need to have .env file in the working directory with these vars in it:
```
DB_HOST=db
DB_PORT=5432
DB_NAME=db
DB_USER=admin
DB_PASSWORD=admin
HTTP_PORT=8080
```

Swagger documentation can be accessed on /swagger/index.html
